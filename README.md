# Pomodoro Timer

## Um App Web de temporizador de Pomodoro. 
### A **Técnica Pomodoro** é um método de gerenciamento de tempo. A técnica consiste na utilização de um cronômetro para dividir o trabalho em períodos de 25 minutos, separados por breves intervalos. [Pomodoro Wiki](https://pt.wikipedia.org/wiki/T%C3%A9cnica_pomodoro)
